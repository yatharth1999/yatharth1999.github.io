Automated Attendance System

This project was designed to streamline attendance tracking using facial recognition technology. The system uses advanced algorithms to detect and recognize faces, logging attendance data efficiently. Key features include:

  • User-friendly web interface for easy registration and management.
  • High accuracy and reliability in face detection and recognition.
  • Integration with backend systems for real-time data updates.

Tools Used: Python, OpenCV, Flask, SQL
