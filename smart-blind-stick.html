Smart Blind Stick

The Smart Blind Stick project aimed to improve mobility for visually impaired individuals. Using Arduino technology and ultrasonic sensors, the stick detects obstacles and provides feedback to the user. Key features include:

  • Obstacle detection with ultrasonic sensors.
  • Emergency contact feature for added safety.
  • Lightweight and easy to use design.

Tools Used: Arduino, Ultrasonic Sensors, C/C++
